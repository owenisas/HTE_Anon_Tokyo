How It Works

🔍 Layer 1 — User Level

The Chrome extension detects zero-width Unicode watermarks embedded in AI-generated text. Any user can scan web pages for hidden metadata without needing an account.

🏢 Layer 2 — Company Level

Authorized companies receive an ECDSA private key. They watermark responses, sign the hash, and anchor it to the provenance chain. Only valid signatures create new blocks.
